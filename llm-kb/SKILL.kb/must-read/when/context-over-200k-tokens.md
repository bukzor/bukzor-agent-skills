# When Context Exceeds 200k Tokens

Empirically: 200k usually means crossing 300k before wrap. Verbatim
user signals decay first.

## Step 1: Case-study-worthy?

Was the session producing reusable lessons?

- Repeated user corrections on the same issue
- Methodology failure (wrong frame held for multiple turns)
- Surprise, dead-end, or unexpected friction
- Pattern or principle worth naming for future agents

If none apply, the session is execution-only. No action.

If case-study-worthy: run `../../procedures.kb/post-mortem.md`.
