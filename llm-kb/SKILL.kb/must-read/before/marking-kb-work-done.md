# Before Marking kb Work Done

You're about to commit, request review, or transition to a new task on
kb content you've created or edited. Run the self-audit sweep before
drift hardens.

**Procedure: `../../procedures.kb/run-self-audits.md`.**

The procedure enumerates audits by urgency tier; treat Tier 1
(blocking) as non-negotiable and Tier 2 as required-when-applicable.
