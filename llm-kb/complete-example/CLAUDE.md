# Birthday Party Planning - Agent Guide

## What This Is

A knowledge base for planning Alex's 8th birthday party on March 15.

## Directory Purposes

- `guests.kb/` - Invited children, RSVP status, dietary restrictions
- `food.kb/` - Food planning (nested `cake.kb/` for cake decisions)
- `games.kb/` - Party activities and games
- `decorations.kb/` - Visual elements and setup
- `timeline.kb/` - Event schedule and logistics

## Cross-Cutting Constraints

- Budget: $200 total
- Space: Backyard (weather permitting) or living room
- Duration: 2:00 PM - 5:00 PM
- Allergy alert: Emma has peanut allergy

## Adding Content

Each directory has its own CLAUDE.md explaining what belongs there.
New items should go in the most specific applicable directory.
