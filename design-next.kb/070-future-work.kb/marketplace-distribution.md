# Marketplace Distribution

Publish the plugin through a Claude Code marketplace (community or
self-hosted `marketplace.json`) so other operators can adopt the
system whole. Deferred because publishing freezes interfaces: names,
genre schemas, and hook contracts should survive a month of personal
use unchanged before strangers depend on them.
