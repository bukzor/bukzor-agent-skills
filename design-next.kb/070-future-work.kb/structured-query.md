# Structured Query

`kb query` grows frontmatter-predicate support (filter by status,
date range, `why:` reachability) only if plain `ls`/`grep` demonstrably
fail recurring needs. Deferred deliberately: v1 proved shell
one-liners cover nearly everything, and a query DSL is exactly the
kind of bespoke surface `native-first` warns about. The trigger's
three-occurrence bar keeps the burden of proof on the DSL.
