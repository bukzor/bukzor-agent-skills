# Skills Over MCP

If the MCP Skills Extension graduates from draft, serving the
system's skills and class schemas over MCP would give remote
consumers (other machines, teammates, CI) the same source of truth
the local plugin uses — and would revisit `cross-reference-notation`,
since a standardized `skill://` grammar with native resolution changes
that entry's premise. Until then: watch, don't build.
