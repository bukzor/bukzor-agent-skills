# Mechanism Over Exhortation

Behavior that must happen is guaranteed by structure — hooks,
validators, schemas, tools — never by prose emphasis. An IMPERATIVE
block is a defect report: it marks a place where a mechanism is
missing and the system is compensating by shouting.

This is v1's sharpest lesson. The must-read banks, the `WILL FAIL`
warnings, and the triple-stated read requirements all accreted as
patches for non-compliance, and each patch decayed. The platform now
agrees: Claude Code's own docs state that CLAUDE.md and rules are
"context, not enforced configuration" — hooks are the only true gate.

Prose retains exactly one enforcement role: judgment calls that no
mechanism can express. Everything else compiles down.
